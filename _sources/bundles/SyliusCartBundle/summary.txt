Resumen
=======

Referencia de configuración
---------------------------

.. code-block:: yaml

    sylius_cart:
        driver: ~ # El controlador usado por la capa de persistencia.
        engine: twig # Motor de plantillas a usar por omisión.
        resolver: ~ # id del servicio resolutor de elementos del carrito.
        provider: sylius.cart_provider.default # Proporciona el id del servicio Cart.
        storage: sylius.cart_storage.session # El id de almacenamiento del carrito para el proveedor predefinido.
        classes:
            cart:
                model: ~ # La clase del modelo cart.
                controller: Sylius\Bundle\CartBundle\Controller\CartController
                repository: ~ # Aquí puedes sustituir la clase del repositorio.
                form: Sylius\Bundle\CartBundle\Form\Type\CartType # El nombre del tipo del formulario a usar.
            item:
                model: ~ # La clase del model item del carrito.
                controller: Sylius\Bundle\CartBundle\Controller\CartItemController
                repository: ~ # Aquí puedes sustituir la clase del repositorio.
                form: Sylius\Bundle\CartBundle\Form\Type\CartItemType # El nombre de la clase del tipo de formulario a usar.

Ejemplos de `phpspec2 <http://phpspec.net>`_
--------------------------------------------

.. code-block:: bash

    $ composer install --dev --prefer-dist
