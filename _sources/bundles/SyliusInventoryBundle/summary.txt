Resumen
=======

Referencia de configuración
---------------------------

.. code-block:: yaml

    sylius_inventory:
        driver: ~ # El controlador usado por la capa de persistencia.
        engine: twig # Motor de plantillas a usar por omisión.
        backorders: true # Activa/desactiva los pedidos pendientes.
        events: ~ # Arreglo de eventos para InventoryChangeListener
        classes:
            unit:
                model: ~ # La clase del modelo de la unidad del inventario.
                controller: Sylius\Bundle\ResourceBundle\Controller\ResourceController
                repository: ~ # Aquí puedes sustituir la clase del repositorio.
            stockable:
                model: ~ # El modelo de la clase inventariable.
                controller: Sylius\Bundle\ResourceBundle\Controller\ResourceController
                repository: ~ # Aquí puedes sustituir la clase del repositorio.
        checker: sylius_inventory.checker.default # El id del servicio comprobador de disponibilidad.
        operator: sylius_inventory.operator.default # El id del servicio operador del inventario.

Ejemplos de `phpspec2 <http://phpspec.net>`_
--------------------------------------------

.. code-block:: bash

    $ composer install --dev --prefer-dist
