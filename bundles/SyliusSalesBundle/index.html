SyliusSalesBundle¶

Este paquete es el fundamenteo para manejar pedidos en proyectos Symfony2. Te permite utilizar cualquier modelo como mercancías.

También incluye una supercaracterística de flexibles ajustes, la cual sirve como base para cualesquier sistema de impuestos, cargos de embarque o descuentos.
